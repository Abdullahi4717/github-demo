# Prosjektuke-1

Nettbutikk for gamingutstyr(Midlertidig)

Jeg skal lage en enkel nettbutikk som selger gaming utstyr som tastatur, mus og headset.

Teknologi, Utsyr og språk

Html / Css - Frontend 

Figma - Skisse av design


Hvordan jeg viser kompentanse i fagene
Utvikling:
Skrive kode i HTML og CSS
Lager en fungerende nettside

Drift:
Viser hvordan flask serveren setter opp og kjøres

Brukerstøtte:
Enkel dokumentasjon
Tester at nettsiden fungerer og er lett å bruke

Oppgaver i proskjektet:

Skisse:

Lage designskisse i figma
Vise hvor produktene, handlekurven og navigasjon skal være

Frontend:

Lage Index.html med produktvisning
Lage style.css for utseende

Backend:

koble flask til databasen og vise data på nettsiden
legge til eksempeldata( Produkter, priser og bilder)

Testing og dokumentasjon:

Teste at alt fungerer sammen
rette eventuelle feil

Lov om bilder fra power.no

Hvis nettsiden kun brukes internt i skolen (ikke offentlig), kan du bruke bildene
Dette er fordi bruken da faller inn under undervisningsunntak / privat krets, som teksten din beskriver.
Dette er lov så lenge:
•	nettsiden er bare for lærer og elever
•	den ligger på en lukket plattform (Canvas, ItsLearning, lokal server, USB, ZIP-fil levert inn osv.)
•	den er ikke tilgjengelig på internett for alle
Da kan du bruke bildene.
